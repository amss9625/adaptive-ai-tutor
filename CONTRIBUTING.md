# Contributing to Adaptive AI Tutor

Thank you for your interest in contributing! This project is focused on studying **how users learn and reason**.

## How to Contribute

1. **Report Issues**
   - Confusion points in exercises
   - Bugs in the evaluator, watcher, or GUI
   - Suggestions for new exercises or domains

2. **Submit Pull Requests**
   - Fork the repository
   - Make your changes in a separate branch
   - Ensure your changes maintain the experimental integrity of the system
   - Submit a pull request with a clear description

3. **Feedback**
   - Use Issues to report user experience observations
   - Include steps to reproduce any errors
   - Be specific about where a user might get stuck

## Guidelines

- Focus on **learning and reasoning**, not just correctness
- Do **not** submit exercises that give direct answers
- Keep the modular structure in mind (ai/, gui/, data/)

Thank you for helping improve Adaptive AI Tutor!
